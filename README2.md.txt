## THE BASICS

'git init': create a repositroy in your current folder
'git status': tells you everything you need to know about current state
'git add <path>': add the ,path. to the stagin area
'git commit': will open up an editor to write a commit message
	-'git commit -m "your message"'

'git log': will show you git log
  - git log--online': will give you the oneline version on log

  'git diff': diff of current state with previous file
    - git diff -- staged': diff file in the staging area
    	between 2 files

'HEAD': where you are currently looking (the files you are looking at)
